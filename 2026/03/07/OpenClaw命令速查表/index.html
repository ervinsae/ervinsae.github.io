OpenClaw常用命令自查表

发表于 2026-03-07 | 分类于 技术
字数统计: 388 | 阅读时长 ≈ 1

🔍 文档内容摘要

核心 CLI 命令

  • openclaw gateway — 运行 WebSocket 网关服务器
  • openclaw gateway start|stop|restart — 管理网关服务
  • openclaw channels login — WhatsApp 二维码配对
  • openclaw channels add — 添加机器人
  • openclaw channels status --probe — 渠道状态检测
  • openclaw onboard — 交互式设置向导
  • openclaw doctor — 健康检查 + 快速修复
  • openclaw models list|set|status — 模型管理 + 认证状态

快速安装

  • npm install -g openclaw@latest
  • openclaw onboarding --install-daemon
  • 全局标志：--dev、--profile、--json

渠道管理

  • WhatsApp: channels add --whatsapp
  • Telegram: channels add --telegram
  • Discord: channels add --discord
  • Slack: channels add --slack

工作区结构

  • AGENTS.md — 指令
  • SOUL.md — 人格/语调
  • USER.md — 偏好
  • IDENTITY.md — 名称
  • MEMORY.md — 长期记忆
  • BOOT.md — 启动
  • ROOT: ~/.OPENCLAW/WORKSPACE

斜杠命令/COMMANDS

  • /status — 运行状态
  • /model <m> — 切换模型
  • /compact — 压缩上下文
  • /new — 开启新会话
  • /stop — 中止
  • /think — 推理模式

基本路径映射

  • CONFIG: ~/.openclaw/openclaw.json
  • CREDENTIALS: ~/.openclaw/credentials/
  • LOGS: /tmp/openclaw/*.log

语音与 TTS

  • OpenAI / ElevenLabs: Premium
  • Edge TTS: Free (No API Key)
  • messages.tts.auto: "always"

记忆与模型

  • 向量搜索：memory search
  • 模型切换：models
  • 认证设置：models auth

故障排除

  • 没有私信回复 → PAIRING LIST → APPROVE
  • 群组中静默 → MENTIONPATTERNS CONFIG
  • 网关关闭 → DOCTOR --DEEP
  • 记忆错误 → MEMORY INDEX

自动化与研究

  • 浏览器操控：browser start
  • 定时任务：cron list
  • 心跳检查：30m
